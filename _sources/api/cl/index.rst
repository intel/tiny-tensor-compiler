tinytc_cl
=========

OpenCL runtime support is provided in libtinytc_cl.
C users should include the header

.. code-block:: c

   #include "tinytc/tinytc_cl.h"

and C++ users should include the header

.. code-block:: cpp

   #include "tinytc/tinytc_cl.hpp"
