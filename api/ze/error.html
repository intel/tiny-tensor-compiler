Error handling#

All calls to the Level Zero API are wrapped in the ZE_CHECK macro. The macro checks the return code and throws a tinytc::level_zero_error should the return code not equal ZE_RESULT_SUCCESS.

ZE_CHECK(X)#

Checks Level Zero call for error and throws exception if call was not successful.

Parameters:

  • X – Level Zero call or ze_result_t status code
