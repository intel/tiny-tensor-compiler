Runtime#

Here, functions to online compile binaries to native Level Zero types are described as well as the implementation of the runtime abstraction for Level Zero. In most cases, the API documented here needs not be used directly but one just needs to pass the tinytc::level_zero_runtime type to the tinytc::tensor_kernel_bundle class as template argument.
