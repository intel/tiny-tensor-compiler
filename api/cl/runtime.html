Runtime#

Here, functions to online compile binaries to native OpenCL types are described as well as the implementation of the runtime abstraction for OpenCL. In most cases, the API documented here needs not be used directly but one just needs to pass the tinytc::opencl_runtime type to the tinytc::tensor_kernel_bundle class as template argument.

Concepts#

Standard OpenCL does not support Unified Shared Memory but only via the cl_intel_unified_shared_memory extension. Therefore, one needs to handle pointer kernel arguments differently from cl_mem and other kernel arguments.

template<typename T>
concept pointer_kernel_argument#

Check whether T is of pointer type but not of cl_mem type.

value type

template<typename T>
