tinytc_ze
=========

Level Zero runtime support is provided in libtinytc_ze.
C users should include the header

.. code-block:: c

   #include "tinytc/tinytc_ze.h"

and C++ users should include the header

.. code-block:: cpp

   #include "tinytc/tinytc_ze.hpp"
